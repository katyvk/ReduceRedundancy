ReduceRedundancy

In these crazy times, we all want to help - but how if we don't know who's working on what and what they're up to?

Let's increase the efficiency of solution development for Covid19 by connecting people working (or wanting to work) on similar goals!
Find projects working in a space relevant to you so that you can contact and support one another - so that you dont have to reinvent the wheel over and over again.

Here's what to do:
(1) Enter your project, or a project you have heard about (,or both), into the form below.
(2) Filter through the list of projects in the sheet to find relevant projects.
(3) Use the comment function to request to add / change information or to offer support to the project. .. for now, you'll have to use this link to comment to request/suggest collboration, etc: AirTable-Comment

Feedback: If you have feedback for us, please input it into the Feedback Form.
